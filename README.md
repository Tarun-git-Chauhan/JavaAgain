# JavaAgain

This repository documents my journey of relearning Java and rebuilding strong programming fundamentals.

My goal is to master backend development and eventually move into cloud infrastructure and security engineering.

---

## Learning Roadmap

Stage 1 — Java Fundamentals
- Variables
- Loops
- Arrays
- Methods
- Input / Output
- Problem solving

Stage 2 — Object Oriented Programming
- Classes and Objects
- Encapsulation
- Constructors
- Inheritance
- Polymorphism
- Abstraction

Stage 3 — Core Java
- Collections (ArrayList, HashMap, etc.)
- Exception Handling
- File Handling
- Streams and Lambdas

Stage 4 — Data Structures
- Arrays
- Linked Lists
- Stacks
- Queues
- Trees
- Hashing

Stage 5 — Backend Development
- REST APIs
- Spring Boot
- Authentication
- Database design
- PostgreSQL

Stage 6 — Infrastructure and Security
- Linux
- Docker
- Cloud deployment
- Security fundamentals

## Purpose

This repository is not just for storing code but for documenting the process of learning and improving as a developer.

Each folder represents practice exercises, experiments, and concept implementations.
